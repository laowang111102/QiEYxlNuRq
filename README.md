# 酒店管理系统 java1142

## 项目概述

酒店管理系统是一款基于前后端分离架构的信息管理系统，旨在提供便捷、高效的服务，满足酒店日常运营需求。

## 技术栈

- 前端技术：Vue、Element-UI
- 后端技术：Spring Boot、MyBatis Plus
- 数据库：MySQL
- 开发语言：Java

## 功能模块

### 系统角色

- 管理员
- 用户

#### 管理员功能

- 登录
- 用户管理
- 订单管理
  - 处理酒店预约订单
  - 查看订单列表
- 房间管理
  - 房间列表
  - 详情查询
  - 添加房间
  - 修改房间
- 评论列表

#### 用户功能

- 注册
- 登录
- 查询空余房间
- 房间预约
- 我的
  - 查看预约订单

## 运行环境

- 建议使用主流浏览器，例如：Chrome、Firefox、Edge 等。
- 后端需 Java 环境支持。

## 部署步骤

1. 搭建 MySQL 数据库环境。
2. 部署后端服务。
3. 部署前端服务。
4. 根据文档进行系统配置。
